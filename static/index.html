Biothings Crawler

Serving structured metadata with Biothings API for the following datasets in schema.org standard.
Last updated: Jan 2020

  • Omicsdi
  • Zenodo
  • Harvard Dataverse
  • NCBI Geo
  • ImmPort
  • NYU Data Catalog
  • CTSA Data Discovery Engine

Examples:

http://crawler.biothings.io/api/query?q=__any__
http://crawler.biothings.io/api/query?q=virus
